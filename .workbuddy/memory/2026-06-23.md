## 全局浮动 Toast 改造

将所有页面的内联提示消息和 `alert()` 调用改为统一浮动 Toast（通过 `notification` 自定义事件 + AppRouter 全局 Toast 容器）：

| 文件 | 改动 |
|:---|:---|
| Trading.tsx | 移除内联 notification div（`marginBottom: 12px` 导致页面抖动），改用 `toast()` |
| Inventory.tsx | 移除 `useResult` 内联提示和 3 处 `alert()`，改用 `toast()` |
| Sect.tsx | 移除 5 处 `alert()`（藏经阁、丹药堂、任务堂），改用 `toast()` |
| SkillTree.tsx | 移除 `message` 状态和内联消息 div，改用 `toast()` |

Toast 使用 `position: fixed` 顶部居中 + `slideDown` 动画，完全不占用页面布局空间，彻底消除抖动。
